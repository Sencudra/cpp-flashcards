# Contributing to cpp-flashcards

Thank you for your interest in contributing!

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Make your changes**.
3. **Test your changes** to ensure nothing is broken.
4. **Open a pull request** with a clear description of your changes.

## Guidelines

- Keep code and documentation clear and concise.
- For changes to the flashcard deck, ensure `deck.json` remains valid JSON (the CI will check this).
- Add or update documentation as needed.
- Be respectful and constructive in discussions.

## Questions or Suggestions?
Open an issue or start a discussion!
